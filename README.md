## 사지마켓에 오신 것을 환영합니다!

---

🐱 사지마켓은 전국의 지자체 유기동물 보호센터와 연계하여 운영되는 유기묘 무료 분양 플랫폼입니다.

👥 우리는 턱없이 낮은 유기동물 입양률을 해결하기 위해 유기동물 보호소의 접근성을 높이는데 집중합니다.

💬 채팅 기능을 통해 보호센터와 간편하게 문의가 가능합니다.

😻 입양된 냥이들의 귀여운 일상 모습을 자랑하는 SNS 기능도 있답니다!

  - useMemo 사용하여 불필요한 재렌더링 방지
